## 功能说明
 本demo是用kotlin语言，通过系统Keyboard和KeyboardView实现的安全键盘，可转换使用数字键盘、字母键盘和符号键盘三种输入方式。
 通过本demo。可以学到kotlin的基本使用，以及通过系统自带view实现安全键盘的实现方式
